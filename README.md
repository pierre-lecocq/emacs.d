# Emacs configuration

Based on Emacs 30+

## Features

- LSP support, with `eglot`
- Completion, with `company` (+ `company-box`, `company-go`)
- Syntax checking, with `flycheck` (+ `flycheck-eglot`, `flycheck-golangci-lint`)
- Project management, with built-in `project.el`
- Fuzzy minibuffer completion, with `ido` (+ `flx-ido`, `ido-vertical-mode`, `ido-completing-read+`)
- Git integration, with `magit` and `git-gutter`
- Snippets, with `yasnippet`
- Terminal integration, with `vterm`
- Key discovery, with `which-key`
- Code navigation, with `imenu-list` and `which-func`
- EditorConfig support, with `editorconfig`
- Language support: Go, PHP, JavaScript/JSX, TypeScript, SQL, YAML, JSON, Markdown, Dockerfile, Terraform, GraphQL, dotenv
- REST client, with `restclient`
- Custom dark theme, with `simple-modeline`
